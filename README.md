<!-- Run Locally -->

### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/doandroidsdreamof/havas-typescript-case-study.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm dev
```
